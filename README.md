# Consulta-Horario-GUI
Aplicación de escritorio  hecha en Python para consultar horarios académicos
#  Gestor de Horarios Académicos

Un organizador de clases sencillo desarrollado en **Python** utilizando el paradigma imperativo y una interfaz gráfica moderna.

## Características
- Interfaz gráfica moderna con `customtkinter`.
- Consulta de clases por día y hora (formato 24h).
- Manejo de errores para entradas no válidas.
- Modo oscuro (Dark Mode) por defecto.

##  Requisitos
debes instalar la librería en tu terminal o consola de comandos antes de ejecutarlo
Es necesario tener instalado Python y la librería CustomTkinter:
